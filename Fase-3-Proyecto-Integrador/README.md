# Fase 3: Proyecto Integrador - CI/CD Pipeline Multi-Plataforma

## Descripción
Este proyecto implementa un pipeline CI/CD completo para una aplicación Python que incluye pruebas unitarias, generación de logs e interacción con el sistema de archivos.

## Características
- ✅ Aplicación Python simple con pruebas unitarias
- ✅ Sistema de logging multi-nivel
- ✅ Interacción con sistema de archivos
- ✅ Pipeline CI/CD con matriz multi-OS
- ✅ Escaneo de seguridad automatizado
- ✅ Release automático con rollback

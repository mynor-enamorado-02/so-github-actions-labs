# Aplicación SO CI/CD con Flask

Esta es una aplicación web simple creada con Flask Python

## Características

- ✅ Página principal con información de la aplicación
- ✅ 3 rutas/endpoints funcionales:
  - `/` - Página principal
  - `/info` - Información del sistema y variables de entorno
  - `/health` - Verificación de salud del servicio
- ✅ Muestra variables de entorno como `FLASK_ENV`, `NODE_ENV`, `VERSION`
- ✅ Tests unitarios con Pytest

## Requisitos

- Python 3.10+
- Flask
- Pytest
